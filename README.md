#Deskripsi Program
Program ini merupakan aplikasi **Sistem Manajemen Nilai Mahasiswa** berbasis **Python** yang dibuat untuk memenuhi **Project UAS Bahasa Pemrograman**.  
Program dikembangkan menggunakan konsep **Object Oriented Programming (OOP)** dan **modular**, dengan pemisahan class **data**, **process**, dan **view**.

Program berjalan pada **Command Line Interface (CLI)** dan mampu menerima input dari pengguna, melakukan validasi input, serta menampilkan hasil dalam bentuk **tabel**.

---

#Tujuan Program
- Menerapkan konsep **OOP (Object Oriented Programming)**
- Menerapkan konsep **modular programming**
- Menggunakan **validasi input dan exception handling**
- Menampilkan output program secara terstruktur

#Fitur Program
- Input data mahasiswa (Nama, NIM, Nilai)
- Validasi input menggunakan exception
- Penyimpanan data mahasiswa
- Menampilkan data mahasiswa dalam bentuk tabel
- Menu interaktif berbasis teks

#Struktur Program

#Penjelasan File
**data.py**  
  Berisi class `Mahasiswa` yang menyimpan atribut nama, NIM, dan nilai.

**process.py**  
  Berisi class `MahasiswaProcess` yang berfungsi untuk memproses data mahasiswa, melakukan validasi input, dan menyimpan data.

**view.py**  
  Berisi class `MahasiswaView` yang berfungsi menampilkan data mahasiswa dalam bentuk tabel.

**main.py**  
  File utama untuk menjalankan program, menampilkan menu, dan menerima input dari pengguna.

#Cara Menjalankan Program
1. Pastikan Python sudah terinstal
2. Buka terminal / command prompt
3. Masuk ke folder project
4. Jalankan perintah berikut:
bash
python main.py
